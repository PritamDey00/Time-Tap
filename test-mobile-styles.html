Mobile Account Button Test

This page tests the mobile responsiveness of the account button.

The account button should:

  • Always be visible in the top-left corner
  • Not be cut off on any side
  • Respect safe areas on mobile devices
  • Scale appropriately for different screen sizes
  • Maintain proper touch target size (minimum 44px)

Test Instructions:

  1. Open this page on various mobile devices
  2. Test in both portrait and landscape orientations
  3. Check that the button is fully visible
  4. Verify it doesn't overlap with browser UI
  5. Test on devices with notches/rounded corners
